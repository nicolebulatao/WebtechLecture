THREE FUNDAMENTAL TECHNOLOGIES THAT CONTRIBUTES TO THE FOUNDATION OF THE WEB

HTML (Hypertext Markup Language): publishing format for the Web, has the ability to format documents and link to other documents and resources.

URI (Uniform Resource Identifier): address that is unique to each resources on the Web

HTTP (Hypertext Transfer Protocol):Application layer used primarily to retrieve hypertext (on hypermedia) documents and resources on the Web.It is jointly developed by the W3C and the IETF

HTTP VERSION HISTORY

Developed by: W3C and WHATWG

In 1980 Tim-Berners Lee developed the first HTML. Used since 1990.

HTTP 0.9 (1991)

HTTP 1.0 (RFC 1945, May 1996)

  • viewed in the web as a simple text. There was 20 elements included in this version.
  • Composed of SGML that included href tag.
  • Styling of content are very limited because you can not use tables or frames, text fonts, use forms and change the background of pages.

    HTTP 2 (RFC 7540, May 2015)

  • The first official set of HTML standards.
  • Text format are both in machine-readable and human-readable form. Added exception -(META|LINK) to TITLE.

    • HTTP 3.0

  • The draft was introduced by Dave Raggett. Included with new and improved html abilities.
  • Later abandoned due to slow implementation of the improvements.

    • HTTP 3.2 (1997)

  • This version supports table creation and cover a large option for form elements.
  • Allowed mathematical equations in web pages.
  • A specification that supports CSS.

    • HTML 4.01 (1999)

  • 13 of the 20 elements from the html version 1 are still used by html 4.
  • Last iteration of classic HTML.

    • XHTML 1.0 (2000)

  • Joint-standard with HTML 4.01. New Branch of HTML.
  • Machine readable file format.
  • Strict than HTML.
  • It has an expression of HTML but still comply with the syntax of XML and therefore can still be viewed, edited and validated with XML tools
  • XHTML was created to introduce new elements and add element attributes.

    • HTML5

  • The updated version of all the HTML from 1 to 4. This version was made clearer for readability sake.
